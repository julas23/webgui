The objective of this project is not to be another tool for monitoring or administration, but to replace the basic need of direct access to the servers, working like a layer to keep interlaced user and server, without a login and after that a change to another user with necessary privileges.

That application can be executed with the necessary privileges to run all the instructions, tools and commands easily and faster, think about that layer like a Documentention as a WIKI, but a lot smarter, that have not only the information about the Tasks, but the opportunity and condition to run it showing the System STDOUT to check if everything is working fine.

In specific cases when the usual command do not work, you can use the GET space to input an incremental command with parameters, or better, you can change to a WEBSHELL where you have a TRUE Console Command Line, to work in the server you need.

Here is the iframe content, this frame is used basically to receive all information sent by the Operational System redirecting the STDOUT here. The command that will request that STDOUT, must be included in the menu, the frame in left side, and is totally configurable to keep the necessities and most important commands used by the team.

Enjoy it!


O objetivo deste projeto não é ser mais uma ferramenta para monitoramento ou administração, mas para substituir a necessidade fundamental de acesso direto aos servidores, funcionando como uma camada para manter usuário e servidor interagindo, sem um início de sessão para depois uma mudança para outro usuário com os privilégios necessários.

A execução pode ser executada com os privilégios necessários para efetuar todas as instruções, ferramentas e comandos de forma fácil e rápida, pense nesta camada como uma Documentação ou uma WIKI, mas muito mais inteligente, que tem não só as informações sobre as tarefas, mas a oportunidade e condição para executá-lo mostrando o STDOUT do sistema para verificar se tudo está funcionando bem.

Em casos específicos, quando o comando de costume não funcionar, você pode usar o espaço GET para a entrada de um comando incremental com os par√¢metros, ou melhor, você pode mudar para um WebShell onde você tem uma linha de comando VERDADEIRO Console, para trabalhar no servidor que você precisa.

Aqui estará o conteúdo da iframe, este quadro é usado basicamente para receber todas as informações enviadas pelo Sistema Operacional redirecionando o STDOUT aqui. O comando que irá solicitar a saída padrão, deve ser incluído no menu, o quadro no lado esquerdo, e é totalmente configurável para manter as necessidades e comandos mais importantes usados pela equipe.
